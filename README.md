# 🧠 TalentScout AI: Neural Recruitment Engine

**TalentScout AI** is a next-generation, Full-Stack AI platform designed to automate candidate screening using **Llama 3.3** and **RAG (Retrieval-Augmented Generation)**. It features a high-performance "Neural Scanner" for candidates and an Intelligence Dashboard for recruiters.

---

## 🚀 Key Features

* **Neural Resume Scanning:** Real-time PDF analysis using a vectorized RAG engine.
* **Llama 3.3 Intelligence:** Automated scoring (0-100%) and executive summaries of candidate-job fit.
* **Role-Based Access Control (RBAC):** Secure, dedicated environments for Recruiters and Candidates.
* **Gmail Integration:** Automated SMTP notifications for high-match candidate alerts.
* **Neural UI/UX:** A modern, glassmorphic interface built with React, Tailwind CSS, and Framer Motion.

## 🛠️ Tech Stack

### Frontend
* **Core:** React.js (Vite)
* **Styling:** Tailwind CSS (Glassmorphism)
* **Icons:** Lucide-React
* **Routing:** React Router v6 (Protected Routes)

### Backend & AI
* **Neural Engine:** FastAPI (Python) & Node.js (Express)
* **LLM:** Llama 3.3 (via Groq/Ollama)
* **Database:** MongoDB (User & Application Nodes)
* **Vector Store:** FAISS / Pinecone (for RAG retrieval)
